Uygulamayı Çalıştırmak İçin Gerekli Talimatlar
[cite_start]Bu projeyi yerel olarak çalıştırmak için aşağıdaki adımları izleyin[cite: 10]:

Projeyi klonlayın ve dizine gidin:

git clone [SİZİN-GİTHUB-REPO-LİNKİNİZ]
cd ProfileApp
Gerekli bağımlılıkları yükleyin: Proje react-native-safe-area-context [cite: 27] gibi özel Expo paketleri içermektedir. Tüm bağımlılıkları doğru kurmak için npx expo install komutunu çalıştırın:

npx expo install
(Alternatif olarak npm install da kullanılabilir)

Geliştirme sunucusunu başlatın:

npx expo start
Cihazınızda çalıştırın: Terminalde görünen QR kodu, telefonunuzdaki Expo Go uygulaması [cite: 6] ile tarayın. Uygulama otomatik olarak açılacaktır.
